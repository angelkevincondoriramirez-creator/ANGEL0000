Ejemplo 1: Llamada básica con SWI

Este ejemplo muestra cómo una instrucción SWI puede invocar una rutina del sistema para mostrar un mensaje o realizar una operación específica. 

MOV R0, #1      ; Número de sistema (print)
MOV R1, #mensaje
SWI 0           ; Llama a la interrupción de software

La instrucción SWI 0 ejecuta la rutina predefinida en el vector de interrupciones.
