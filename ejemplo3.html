Ejemplo 3: Manejo de excepciones con SWI

En este ejemplo, se genera una excepción mediante SWI para controlar un error en la ejecución del programa. 

MOV R0, #0
CMP R0, #0
BEQ error
SWI 2           ; Llama a la rutina de error

El sistema maneja la excepción y muestra un mensaje de error al usuario.
